# 🪨 📰 ✂️ Rock-paper-Scissors 

A browser version of the game Rock Paper and Scissors

---

### 📸 Preview

*https://haiderrjan.github.io/Rock-Paper-Scissors/*

---

## 🧠 Features

- ➕ **three buttons** - Rock, paper, Scissors
- ✅ **Scores** - Players scores and Ai (Computers score)
- ❌ **Can reset** - via the button called reset
- 🎮 **Up to 5** - Game is best of five

---

## 🧱 Built With

- **HTML5** — Structure of the page  
- **CSS3** - Styling 
- **JavaScript (ES6)** - Event Listner

---

## ⚙️ How It Works

1. Click click any button **(Rock, Paper, Scissors)** 
2. Js will output the result of win or lose 
3. after **five** Games the result of winning or lose will show below 

All interactions are handled dynamically using JavaScript (no page reloads).
